# 描述

> 此插件是根据 tabby 的github官方插件 Tabby-QuickCmd-Plus 基础上进行修改的

# 使用

> 修改代码之后 npm run build 把dist的index.js放入到tabby对应插件下进行替换，重启tabby即可
>
> ```shell
> const tabbyPluginDir = 'C:/Users/iishen717/AppData/Roaming/tabby/plugins/node_modules/
> 						tabby-quick-command-plus/dist'
> 
> #目前此项目的构建路径已经定向到替换位置tabbyPluginDir，npm run build 重启tabby就行
> #如果 Tabby 插件目录不存在了，会自动回退输出到本项目的 dist/ 目录
> ```
