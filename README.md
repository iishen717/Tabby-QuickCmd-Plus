# Tabby-QuickCmd-Plus

Tabby 快捷命令管理插件。浮动面板 + 分组管理 + 多步执行，适配毛玻璃质感。

## 功能

- **命令管理**：名称、命令内容、分组、快捷键、参数变量
- **多步执行**：支持多步骤命令，每步独立延迟，支持 Ctrl+C 中断步骤
- **变量参数**：`{{ paramName }}` 占位符，执行时弹出参数输入框
- **分组管理**：手风琴折叠、新建/重命名/删除分组、拖拽排序
- **快捷栏**：最多固定 3 个常用命令到顶部
- **复制功能**：一键复制命令文本（多步骤只复制第一条）
- **面板操作**：拖拽移动、8 方向调整大小、最小化/关闭
- **首页入口**：支持在 Tabby 首页直接唤出独立面板
- **位置记忆**：按 SSH Profile 记忆面板位置、分组展开状态、滚动位置
- **双入口模式**：工具栏按钮 / 可拖拽浮动按钮，实时切换
- **发送模式**：仅发送到命令行 / 发送并执行 / 多步合并为 `&&` 拼接
- **6 套毛玻璃预设**：浅色/深色/海洋蓝/森林绿/日落橙/薰衣草紫，支持自定义配色
- **中英双语**：自动检测语言，可手动切换
- **数据管理**：JSON 导入/导出，一键清空

## 安装

### 方式一：npm 安装

```bash
npm install tabby-quick-command-plus
```

### 方式二：手动安装

1. 将本项目复制到 Tabby 插件目录：

```
Windows: %APPDATA%/tabby/plugins/node_modules/tabby-quick-command-plus/
macOS:   ~/Library/Application Support/tabby/plugins/node_modules/tabby-quick-command-plus/
Linux:   ~/.config/tabby/plugins/node_modules/tabby-quick-command-plus/
```

2. 确保目录下包含 `package.json` 和 `dist/index.js` 两个文件。

3. 重启 Tabby。

### 方式三：从源码构建

```bash
git clone <repo-url>
cd Tabby-QuickCmd-Plus
npm install
npm run build
```

构建产物在 `dist/` 目录，将整个项目复制到上述插件目录即可。

## 使用

### 打开面板

- **终端内**：点击终端工具栏或右上角的 `>_ QC+` 按钮
- **首页**：点击首页的 "QuickCmd-Plus" 按钮直接弹出独立面板

### 添加命令

1. 点击面板底部 **"+ 新建命令"** 或分组标题右侧的 **"+"**
2. 填写命令信息：
   - **名称**：命令的显示名称
   - **命令内容**：支持多步骤，每步独立延迟
     - 点击 **"+ 添加命令"** 增加命令步骤
     - 点击 **"+ 添加中断"** 插入 Ctrl+C 中断步骤
     - 拖拽 `≡` 手柄调整步骤顺序
   - **分组**：选择或新建分组
   - **发送并执行**：勾选后自动追加换行执行
   - **固定**：显示在面板顶部快捷栏
3. 点击 **"保存"**

### 执行命令

- **单击**命令行：发送到终端（不自动执行）
- **双击**命令行：打开编辑弹窗
- 悬停后右侧操作按钮：

| 按钮 | 功能 |
|------|------|
| ❐ | 复制命令文本（多步只取第一条） |
| ✎ | 编辑命令 |
| ⇧ | 仅发送到命令行（不执行） |
| ⏎ | 发送并执行 |
| ✕ | 删除命令 |

### 多步命令

每个命令可包含多个步骤，每步支持：
- **命令步骤**：普通命令文本
- **中断步骤**：发送 Ctrl+C
- **独立延迟**：每步执行后的等待时间（秒）

多步命令发送时，可通过弹窗选择：
- **确定**：将所有步骤用 `&&` 拼接为一行发送
- **取消**：逐条发送到命令行

### 变量参数

命令文本中使用 `{{ 变量名 }}` 格式定义参数，执行时弹出输入框填入实际值。例如：

```bash
ssh {{ user }}@{{ host }}
```

### 管理分组

- 点击分组标题 **展开/折叠**命令列表（手风琴模式）
- 分组右侧 **"+"**：在该分组下新建命令
- 分组右侧 **"✎"**：重命名分组 / 调整排序
- 分组右侧 **"−"**：删除分组（命令移到未分组）

### 快捷栏

编辑命令时勾选 **"固定"**，该命令将出现在面板顶部的快捷栏中（最多 3 个）。

### 主题设置

打开 Tabby 设置 → 左侧 **"QuickCmd+"** 选项卡：
- **界面语言**：自动 / 中文 / English
- **颜色主题**：6 套毛玻璃预设 + 自定义配色
- **字体大小**：11-18px 滑块调节
- **入口模式**：工具栏按钮 / 浮动按钮

### 数据备份

设置页 **"数据"** 区域：
- **导出**：下载 JSON 备份文件
- **导入**：选择 JSON 文件恢复数据
- **清空**：输入 `DELETE` 确认清除所有数据
